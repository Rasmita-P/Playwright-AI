# Playwright AI & LLM for QA

Welcome to the **Playwright AI** project! This repository is dedicated to exploring the intersection of Large Language Models (LLMs) and Quality Assurance (QA) workflows, specifically focusing on integration with Playwright.

## 🚀 Project Overview

This project provides a foundation for SDETs and QA engineers to understand and implement AI-driven testing strategies. It includes comparisons of various LLMs, practical setup guides, and utility scripts for environment verification.

- **Author:** Rasmita Pradhan
- **Learning Website:** [The Testing Academy](https://thetestingacademy.com/)
